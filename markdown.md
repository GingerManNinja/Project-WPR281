# HIGH LEVEL REQUIREMENTS
- Create issues
- Assign issues
- View all issues
- View a particular issue
- Edit issues
- Create people
- Create projects

### New Ticket Requirements
- Summary of the issue that was discovered
- Detailed description of the issue
- Who identified the issue
- The date on which the issue was identified
- Which project the issue is related to
- Who the issue is assigned to
- A status of the issue [open/ resolved/ overdue] 

### Ticket Update Requirements
- Priority of the issue [ low/ medium/ high]
- Target resolution date
- Actual resolution date
- Resolution summary

# LOW-LEVEL REQUIREMENTS

### Assigning an Issue
- A person must be selected to fix the issue
- Issues can be created first and assigned later

### View a Single Issue
- Display all details of a selected issue
- Refer to "Creating an Issue" for required fields

### Editing an Issue
- Allow updates to:
  - Summary
  - Detailed description
  - Other issue fields

- Bonus:
  - View issues grouped by project

# Creating People
- ID
- Name
- Surname
- Email address
- Unique username

# DATA REQUIREMENTS

### Projects
- List of all current projects

### People
- Information about users who can be assigned issues

### Bugs / Issues
- Full issue details
- Linked project
- Assigned person
